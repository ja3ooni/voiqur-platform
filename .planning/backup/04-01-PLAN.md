# Plan 04-01: ElevenLabs SDK Integration

**Goal:** Implement cloud-based TTS synthesis using the ElevenLabs SDK in `XTTSv2ModelManager`.

**Requirements:** TTS-01, TTS-04, TTS-05
**Success Criteria:**
1. `XTTSv2ModelManager.synthesize()` returns real audio bytes from ElevenLabs API
2. ElevenLabs client is initialized from `ELEVENLABS_API_KEY`
3. Audio output is converted from PCM 16-bit to float32 numpy array
4. Tests in `test_tts_phase4.py` for TTS-01 and TTS-04 pass (un-xfail)

---

## Tasks

### 1. Install Dependencies
- Install `elevenlabs>=2.40.0`

### 2. Implement ElevenLabs Client in `XTTSv2ModelManager`
- Update `initialize()` to create `ElevenLabs` client using `os.getenv("ELEVENLABS_API_KEY")`
- Add `_get_elevenlabs_voice_id()` mapping helper
- Replace mock sine-wave generation in `synthesize()` with `client.text_to_speech.convert()`
- Convert raw bytes to float32 numpy array (normalized -1.0 to 1.0)

### 3. Update Tests
- Remove `xfail` from `test_elevenlabs_synthesis`
- Mock ElevenLabs SDK for unit testing
- Verify synthesis output shape and dtype

### 4. Verification
- Run `pytest tests/test_tts_phase4.py::test_elevenlabs_synthesis -x`
